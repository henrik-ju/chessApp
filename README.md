# Chess App

## Beskrivning
En Schackapp där vänner kan spela mot varandra utan tidsbegränsning. Där man även kan chatta med varandra.

## Hur appen fungerar
Appen använder Firebase på backend för speltillstånd och chatt.

## 
1. Skapa ett konto med mejl och lösenord.
2. Starta ett schackparti eller gå med i ett part skapat av någon annan.
3. Spela och chatta med varandra samtidigt med en krypterad chatt om du vill.

## Betyg

(3p) Using another device. Man kan spela på olika mobiler mot varandra över internet, backend använder Firebase.
(3p) End-to-end encryption. Chatt med end-to-end encryptation som använder AES kryptering.
(4p) Runtime configuration changes. Använder viewModel.
(1p) Different type of accounts. Användare kan navigera som gäst eller skapa konto. Finns även adminkonto
(1p) Loading indicators. Enkel laddningsindikator när appen startar eller laddar något.
(1p) Uploading and downloading images. Användare kan ladda upp bild.
(?p) Own suggestions. Pawn promotion så har en promotionScreen som kommer upp i spelet där användare kan välja pjäs. Spelet använder FEN (Forsyth-Edwards-Notation) via Firebase.
